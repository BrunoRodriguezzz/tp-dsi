MetaMapa

Sistema de código abierto para la recopilación, visualización y mapeo colaborativo de información. Diseñado para maximizar la disponibilidad y veracidad de la información, protegiendo la identidad de quienes la comparten.

Nuestro Propósito

Únete a la Comunidad

Contribuye con información relevante y ayuda a construir una base de datos colaborativa que beneficie a toda la sociedad.
